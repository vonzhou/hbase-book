# 运行《HBase权威指南》书中代码
---

## 前提

搭建好HBase运行环境。


## 设置环境变量

设置MAVEN_HOME, HBASE_CONF_DIR环境变量：

```
export MAVEN_HOME=/usr/share/maven
export HBASE_CONF_DIR=$HBASE_HOME/conf
```
